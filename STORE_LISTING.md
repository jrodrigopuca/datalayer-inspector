# Chrome Web Store Listing

## Short Description (132 characters max)
Inspect, validate, and export GTM dataLayer events in real-time. Built for developers who need more than just a viewer.

## Detailed Description

**Strata — The Developer's DataLayer Inspector**

Stop guessing if your GTM dataLayer is working correctly. Strata gives you real-time visibility into every dataLayer.push() call, with powerful validation and export tools built for developers and technical analysts.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🔍 REAL-TIME EVENT CAPTURE**

• Intercepts all dataLayer.push() calls as they happen
• Color-coded event categories (GTM, Ecommerce, Custom, Errors)
• Multi-container GTM support
• Pause and resume recording anytime
• Search and filter through captured events

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**✅ SCHEMA VALIDATION**

Define expected event structures and validate automatically:

• Type placeholders: @string, @number, @boolean, @array, @object
• Optional fields: @string? won't fail if missing
• Enum validation: @enum(USD, EUR, GBP)
• Auto-generate schemas from captured events
• Import/export schemas as JSON files

Perfect for catching tracking issues before they reach production.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**📤 EXPORT OPTIONS**

• JSON: Export events for debugging or documentation
• Test Assertions: Generate Playwright or Cypress test code
• Evidence: Create PNG or PDF screenshots for QA reports

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**⌨️ DEVELOPER EXPERIENCE**

• Keyboard shortcuts (Alt+Shift+D to toggle)
• JSON tree view with syntax highlighting
• Copy events to clipboard
• Quick popup view from toolbar

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**🔒 PRIVACY FIRST**

Strata operates entirely locally. No data is collected, transmitted, or stored externally. Your dataLayer events never leave your browser.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**HOW TO USE**

1. Open Chrome DevTools (F12)
2. Go to the "Strata" tab
3. Interact with the page to capture events
4. Create schemas to validate structure
5. Export as needed

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

**WHO IS THIS FOR?**

• Frontend developers implementing GTM tracking
• Technical analysts validating data layer implementations
• QA engineers testing analytics setups
• Marketing technologists debugging tag configurations
